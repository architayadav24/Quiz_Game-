Quiz Game Web Application
# OVERVIEW:-

Quiz Game is a web-based application that allows users to test their knowledge through multiple-choice questions. 
The game provides instant feedback, score tracking, and an interactive user experience.

# FEATURE:
Multiple-choice quiz questions
Real-time score calculation
Interactive user interface
Final score and performance summary

TECHNOLOGIES USED:
HTML5
CSS3
JavaScript

# How to Run the Project:

Download or clone the repository:
git clone https://github.com/your-username/quiz-game.git
Open the project folder.
Launch index.html in your browser.
How to Play
Start the quiz.
Read each question carefully.
Select the correct answer from the available options.
Move to the next question.
View your final score after completing the quiz.
